# Pandoc Quick Reference

## Common Input Formats

| Format       | Flag            |
|--------------|-----------------|
| Markdown     | `-f markdown`   |
| HTML         | `-f html`       |
| LaTeX        | `-f latex`      |
| DOCX         | `-f docx`       |
| EPUB         | `-f epub`       |

## Common Output Formats

| Format       | Flag            |
|--------------|-----------------|
| PDF          | `-o output.pdf` |
| HTML         | `-o output.html`|
| DOCX         | `-o output.docx`|
| EPUB         | `-o output.epub`|
| LaTeX        | `-o output.tex` |
